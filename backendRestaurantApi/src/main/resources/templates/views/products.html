Products(resource groups)

Product Schema

Key Type Description
idProduct long The Id of the product
category Category The Category of the product
nameProduct string Name of the product
priceProduct long Price of the product
availableProduct boolean If the product is available
description string The description of the product
image string The description of the product

Use the /products resource to get, create, update or delete products
