Orders(resource groups)

Order Schema

Key Type Description
idOrder int The Id of the order
user (object) The user that owns the order
payment array Details about the order payment
orderDetails array Items that the order have
statusOrder string The STATUS of the order
description string The description of the order
totalPrice long The amount of the order

Use the /order resource to get, create, update or delete orders

List of orders

GET /api/order

Get a single order

GET /api/order/{idOrder}

Get order by user id

GET /api/order/users{idUser}

Save order

POST /api/order

Request body

iduser int required

The id of the user who owns the order

status long required

The status of the order

subtotal long required

The amount of the order

orders array required

The items of the order

create_time string required

The creation time of the order

payment_method int required

The id of the availables payment methods

Update order status

PATCH /api/order/{idOrder}/status

Request body

status string required

The new status order
