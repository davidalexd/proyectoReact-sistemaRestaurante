Users and roles

User Schema

Key Type Description
idUser int The Id of the user
roles array The roles that the user have
username string The username
email string The email of the user
createdAt string Timestamp when was created the user
updatedAt string Timestamp when was updated the user

Use the /users resource to get, create, update or delete users

List of users

GET /api/users

Get a single user

GET /api/users/{idUser}

Profile image

GET /api/users/{idUser}/image

Save profile image

POST /api/users/{idUser}/image

Request body

file MultipartFile required

The profile image to be saved

Total users

GET /api/users/summary

Profile Schema

Key Type Description
idProfile int The Id of the profile
user (object) The user that the profile belongs
firstname string The firstname of the profile
lastname string The lastname of the profile
profilePicture string The profile picture of the profile
phoneNumber string The phone number of the profile
address string The address of the profile

User profile

firstname string required

phoneNumber string required

address string required
