Payments methods(resource groups)

Payment Schema

Key Type Description
idPaymentMethod int The Id of the payment method
namePaymentMethod string The name of the payment method
description string Details

Use the /paymentmethods resource to get, create, update or delete payment methods

List of payment methods

GET /api/paymentmethods

Create payment method

Request body

namePaymentMethod string required

The name of the payment method

Update a payment method

PUT /api/paymentmethods/{idPaymentMethod}

Request body

namePaymentMethod string required

The name of the payment method
