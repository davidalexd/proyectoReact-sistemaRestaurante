Auth(resource groups)

Use the /api/auth resource to signin, register or reset password

Sign in Schema

Key Type Description
username string The username to signin
password string The password to signin
POST /api/auth/signin

Request body

username string required

The username to signup

password string required

The password to signup

Signup Schema

Key Type Description
username string The username to register
email string The password to register
roles array An array of roles, can be admin, mod or empty(default user)
password string The password to signin
POST /api/auth/signup

Request body

username string required

The username to signup

email string required

The password to signup

roles array required

The posible values are:

  • admin

    The role admin can administrate an add moderator

  • mod

    The role mod can administrate

  • empty or default

    The default role is user

password string required

The password to signup

Reset Password

POST /api/auth/{id}/password

Request body

oldPassword string required

The old password

password string required

The new password

passwordConfirmation string required

The password confirmation
